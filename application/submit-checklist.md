# 申报提交前检查

- [ ] 把所有 `YOUR_GITHUB_USERNAME` 替换成你的账号
- [ ] 把 `yourname/moon-layout-core` 替换成你的 MoonBit 模块名
- [ ] 创建 GitHub 仓库并公开
- [ ] 创建 Gitlink 仓库并同步
- [ ] 按 `COMMIT_PLAN.md` 拆出 10-20 个有效 commits
- [ ] 本地运行 `moon check` 和 `moon test`
- [ ] 生成一页 PDF 申报书
- [ ] 在申报书中填写姓名、联系方式、GitHub、Gitlink
- [ ] 提交申报问卷 / 赛事群要求的材料
